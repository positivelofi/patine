# patine

Synthétiseur browser-based conçu pour la musique lofi, ambient et contemplative.

**[→ Ouvrir patine](https://positivelofi.github.io/patine/)**

## Moteurs sonores

- **pat·in** — synthèse soustractive (Moog Ladder, KS, piano additif)
- **emil·in** — synthèse modulaire avec réseau de modulation 10×10

## Contrôles

Conçu pour le jeu expressif au clavier ou via MIDI.  
Pas de presets figés — chaque paramètre est modulable en temps réel.

## Technique

HTML/CSS/JS — aucune dépendance externe.  
AudioWorklet pour le DSP sample-accurate.
